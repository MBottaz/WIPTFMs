# WIPTFMs
Where I Put The Fucking Modules is a project i develop to help my parents decide in which roof section is more convinet to place PV modules

## Things you need to take care of before getting started

Copy all the files.
The MCP server has access to 'data' and 'input' folders, used for:
- input: is where you can put the input files.
- data: is the working directory of the server, it's  recommended to avoid editing the files inside this folder.
